# Riddle GenUI

A Flutter project that uses Gemini to generate riddles.

## Getting Started

This project is a Flutter application that uses the `genui` and `genui_firebase_ai` packages to generate riddles based on user input.

### Prerequisites

- Flutter SDK
- Firebase account and a new Firebase project

### Installation

1. Clone the repository.
2. Set up a new Firebase project and add your `firebase_options.dart` file to the `lib/` directory.
3. Run `flutter pub get` to install the dependencies.
4. Run the app using `flutter run`.
